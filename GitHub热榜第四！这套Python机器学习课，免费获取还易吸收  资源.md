## 里面都有啥？

“容易理解”的flag绝对不是白立的，从课程设置上就能看出，里面的内容绝对友好了。

这套资源主要在解决四个问题：

1. 机器学习的定义是什么？
2. 什么时候开始流行，发展趋势是什么？
3. 机器学习的类别都有哪些？
4. 最常见的机器学习算法以及如何实现它们？

针对这些问题，这套课程的设置了如下目录，共分为三章：

前言

- 机器学习概览

第一章：核心概念

- 交叉验证
- 线性回归
- 过拟合和低度拟合
- 正则化

第二章：监督学习

- 逻辑回归
- 朴素贝叶斯分类
- 决策树
- K-近邻
- 线性支持向量机

第三章：监督学习

- 聚类
- 主成分分析
- 第四章：深度学习
- 多层感知机
- 卷积神经网络
- 自编码器
